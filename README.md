# Текстовый квест — Java веб‑приложение

Интерактивная текстовая игра с ветвлением сюжета, реализованная на Java с использованием сервлетов, JSP и JSTL.

## Технологии

* Java 11+
* Servlet API 4.0
* JSP (JavaServer Pages)
* JSTL (JSP Standard Tag Library)
* Maven 3+ (для сборки)
* Apache Tomcat 9 (сервер приложений)

## Требования к окружению

Перед запуском убедитесь, что у вас установлены:

1. **Java Development Kit (JDK) 11 или выше**
2. **Apache Maven 3.6.0 или выше**
3. **Apache Tomcat 9.0.x** (рекомендуемая версия: 9.0.70+)
4. Веб‑браузер для доступа к интерфейсу игры
